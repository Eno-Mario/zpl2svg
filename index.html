Realtime ZPL to SVG Converter

Source code: https://github.com/Jozo132/zpl2svg - Author: Jozo132

Zebra Programming Language (ZPL) to Scalable Vector Graphics (SVG) converter written in vanilla JS.

The only dependencies for this project are bwip-js for barcode rendering and canvas for drawing graphic fields (only needed for Node.JS)

ZPL example code represents the VDA 4902 standard label format for testing.
